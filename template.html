📚 Supported Element Types

text - Render text with custom fonts and styles
qrcode - Generate QR codes with error correction
image - Display images with custom dimensions and dithering support
rect - Draw rectangles with optional rounded corners
circle - Draw circles or rings
ellipse - Draw ellipses with rotation
line - Draw straight or dashed lines
polygon - Draw polygons with multiple points
path - Draw SVG-like paths

💡 All elements support {{variable}} substitution for dynamic content

🔧 System Variables

Built-in variables that are automatically available in all templates:

📅 Date Variables

{{date.year}} → 2025
{{date.month}} → 11
{{date.month.name}} → 11月
{{date.day}} → 7
{{date.weekday}} → 4 (0=Sunday)
{{date.weekday.name}} → 星期四
{{date.full}} → 2025年11月7日

⏰ Time Variables

{{time.hour}} → 14 (24-hour)
{{time.hour.12}} → 2 (12-hour)
{{time.minute}} → 30
{{time.second}} → 45
{{time.ampm}} → PM
{{time.ampm.cn}} → 下午
{{time.hm}} → 14:30
{{time.hms}} → 14:30:45

⚙️ Other Variables

{{timestamp}} → 1699363845000
Milliseconds
{{timestamp.seconds}} → 1699363845
Unix timestamp
{{canvas.width}} → 400
{{canvas.height}} → 300
💡 Custom Variables: Define your own variables in the "variables" section of your template JSON, e.g., {"variables": {"name": "John"}} then use {{name}}
